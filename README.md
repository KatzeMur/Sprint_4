# qa_python
Написала тесты для класса BooksCollector.

1. Добавление книг (add_new_book) — проверила, что книги с валидным именем добавляются, а с пустым — нет
2. Установка жанра (set_book_genre) — проверила, что жанр устанавливается правильно
3. Получение жанра (get_book_genre) — проверила, что метод возвращает правильный жанр
4. Поиск книг по жанру (get_books_with_specific_genre) — проверила, что метод находит книги нужного жанра
5. Книги для детей (get_books_for_children) — проверила, что книги с возрастным рейтингом не попадают в список
6. Избранное (add_book_in_favorites, delete_book_from_favorites) — проверила добавление и удаление
7. Список избранного (get_list_of_favorites_books) — проверила, что метод возвращает правильный список
